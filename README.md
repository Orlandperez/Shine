# Portafolio — Resumen técnico

Proyecto personal: portafolio web desarrollado con React y Vite.

Tecnologías principales:
- Lenguajes: HTML, CSS, JavaScript (ES6+), JSX
- Framework: React
- Herramienta de desarrollo: Vite

Librerías y utilidades incluidas:
- `react`, `react-dom`
- `react-router-dom` (ruteo)
- `react-hook-form` (formularios)
- `@vitejs/plugin-react` (integración React + Vite)
- ESLint y plugins relacionados (estilo y hooks)

Comandos rápidos:
- Desarrollo: `npm run dev`
- Build de producción: `npm run build`
- Linter: `npm run lint`

Este README es intencionalmente breve; para más detalles, revisa el `package.json`.
